# Utah Lake Bloom Clustering — Science Fair
Short description: K-Means clustering of water-quality to study algal bloom severity.

Files:
- analysis.ipynb — main notebook with code and plots
- data/results.csv — cleaned data (no personal info)
- figures/figure1.png — timeline figure
How to run:
1. Install Python 3 and required libraries (`pip install -r requirements.txt`)
2. Open `analysis.ipynb` in Jupyter Notebook
